# KsqlDB API Client

Simple KsqlDB JavaScript client leveraging observables and promises

## Getting Started

### Install
```
npm i -S ksqldb-rx-client
```

### Import
```
import { KsqlDBClient } from "ksqldb-rx-client";

const ksqlDBClient = new KsqlDBClient("http://localhost:8088");
```
